## Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd d:\WEB
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The optimized build will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## How to Use

1. **Enter Food Details**: Type in the food name and its macros (protein, carbs, calories)
2. **Set Quantity**: Specify how many servings/portions you had
3. **Add Food**: Click "Add Food" button to log it
4. **Track Totals**: Watch your daily totals update automatically
5. **Remove Items**: Click "Remove" to delete a food from your list
